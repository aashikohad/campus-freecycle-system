Campus Freecycle System

The Campus Freecycle System is a web-based platform designed to encourage sustainable practices within a college campus. It allows students and staff to share unused or unwanted items with others instead of discarding them. By promoting the reuse of resources, the system helps reduce waste and fosters an eco-friendly and collaborative campus community.

This platform provides a simple and efficient way for users to post items they no longer need and allow others to claim them for free.

Key Features
1. User Registration and Authentication

Users can create an account and securely log in to access the platform. This ensures that only authorized campus members can post or request items.

2. Post Items for Giveaway

Users can list items they wish to give away by providing details such as item name, description, and contact information.

3. Browse Available Items

Users can explore a list of available items posted by other campus members and find items they may need.

4. Contact Item Owners

Interested users can directly contact the owner of an item to arrange pickup or further communication.

5. User-Friendly Interface

The system provides a clean and intuitive interface that makes it easy for users to navigate, post items, and view listings.

Installation and Setup

Follow these steps to run the project locally:

1. Clone the Repository
git clone https://github.com/your-username/campus-freecycle-system.git
2. Navigate to the Project Directory
cd campus-freecycle-system
3. Install Dependencies
npm install
4. Start the Application
npm start
5. Access the Application

Open your browser and visit:

http://localhost:3000
